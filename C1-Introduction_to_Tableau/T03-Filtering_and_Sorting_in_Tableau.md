## 1. Filtering and Sorting 🔍📊  
Welcome to Chapter 2, where we cover the basics of building and customizing visualizations.

We will focus on:
- 🔍 Filtering  
- 🔃 Sorting  

---

## 2. Filtering 🔍  
Filtering is a key step in creating visualizations (you’ve already used it in Chapter 1).

It involves deciding:
- ✅ What data to keep  
- ❌ What data to exclude  

### You can filter by:
- 🏷️ Category  
- 📅 Date range  
- 📍 Location  
- 🔢 Minimum/maximum values  

---

## 3. Types of Filters in Tableau ⚙️  
Filtering happens at different stages of the Tableau workflow.

### 📌 Order of Operations:
1. 🗂️ Extract filters  
2. 📦 Data source filters  
3. 🧠 Context filters (advanced, not covered here)  
4. 📊 Worksheet filters (focus of this course)

⚠️ Order matters, especially when retrieving top records.

---

## 4. Dimension Filters (🔵 Blue Fields) 📂  
Dimensions are categorical fields.

Filtering dimensions means:
- 🧾 Selecting categories to include or exclude  

### Examples:
- Room type  
- Neighborhood  

### Advanced options:
- 🔎 Wildcard filters (match text patterns)  
- 🎯 Condition-based filters (based on other fields)  
- 🏆 Top/Bottom filters (rank-based filtering)

---

## 5. Measure Filters (🟢 Green Fields) 📏  
Measures are numeric values.

Filtering measures means working with:
- 🔢 Ranges of values  
- ❓ Null or non-null values  

### Examples:
- Price range  
- Number of reviews  
- Minimum nights  

---

## 6. Sorting 🔃  
Sorting helps organize data in a meaningful way.

### Default behavior:
- 🔤 Alphabetical sorting (for dimensions)

### Better alternative:
- 📊 Sort by metric values (ascending or descending)

### Example:
- Sorting products by gross profit is more useful than alphabetical order
