# 🗺️ Hands-On Activity: Mapping Your Data in Tableau

---

# 🎯 Learning Objectives

After completing this activity, participants should be able to:

- Understand geographic data in Tableau
- Identify geographic roles and geocoding
- Recognize the Globe icon (🌐)
- Create Symbol Maps and Filled Maps
- Use Size and Color in maps
- Build interactive geographic visualizations
- Analyze population-based indicators geographically

# Exercise 1: Understanding Geographic Fields 🌍

## Task

Open Tableau Public and connect to **Gapminder_WHO_Population.xlsx**.

### Steps

1. Load the dataset in Tableau.
2. Go to a new worksheet.
3. Locate the **Country** field in the Data Pane.

### Observation

Look for the:

🌐 Globe Icon

This indicates Tableau recognizes the field as geographic data.

### Discussion

Why is geocoding important?

---

# Exercise 2: Create a Symbol Map 📍

## Business Question

> Where are the countries with high population located?

### Steps

1. Double-click **Country**.
2. Tableau automatically generates a map.
3. Drag **Population 2008** to Size.
4. Drag **Growth Rate (%)** to Color.

### Expected Output

A Symbol Map where:

- Circle size = population
- Circle color = growth rate

### Interpretation

- Which country has the largest population?
- Which region shows high growth?

---

# Exercise 3: Understanding Geocoding 🌐

## Task

Check how Tableau assigned coordinates.

### Steps

1. Hover over any country point.
2. Observe Latitude and Longitude values.

### Discussion

What is geocoding?

Answer:

> Converting country names into latitude and longitude coordinates.

---

# Exercise 4: Create a Filled Map 🗺️

## Business Question

> How does population growth vary by country?

### Steps

1. Drag **Country** to the view.
2. Change Marks type to **Filled Map**.
3. Drag **Growth Rate (%)** to Color.

### Expected Output

Countries are shaded based on growth rate.

### Interpretation

- Which regions show higher growth?
- Which regions show low growth?

---

# Exercise 5: Use Region for Aggregation 🌎

## Business Question

> Which region has the highest total population?

### Steps

1. Drag **Region** to Columns.
2. Drag **Population 2008** to Rows.
3. Change aggregation to **SUM**.
4. Convert visualization to **Bar Chart**.

### Discussion

Compare regional population totals.

---

# Exercise 6: Map with Dual Encoding 📍🎨

## Business Question

> Can we compare population size and growth together?

### Steps

1. Create a Symbol Map.
2. Drag:
   - Population 2008 → Size
   - Growth Rate (%) → Color
   - Country → Label

### Expected Output

A rich interactive map with multiple data encodings.

### Insight

What does a large red circle represent?

---

# Exercise 7: Add Filters 🎛️

## Business Question

> Can we focus on a specific region?

### Steps

1. Drag **Region** to Filters.
2. Select only:
   - Asia
3. Click OK.
4. Show Filter.

### Activity

Switch between regions:

- Asia
- Europe
- North America
- South America

### Discussion

How does regional filtering improve analysis?

---

# Exercise 8: Map Tooltips 💬

## Task

Enhance map readability.

### Steps

1. Hover over a country.
2. Click **Tooltip** in Marks card.
3. Edit tooltip:

```text
Country: <Country>
Population (2008): <Population 2008>
Growth Rate: <Growth Rate (%)>
```

### Expected Output

More informative map interactions.

---

# Exercise 9: Sorting Geographic Insights 🔃

## Business Question

> Which countries have the highest population growth?

### Steps

1. Create a bar chart.
2. Drag Country → Rows.
3. Drag Growth Rate (%) → Columns.
4. Sort descending.

### Discussion

Why combine maps and bar charts?
