# 📊 Hands-On Activity: Tableau Interface & Data Roles

---

# 🎯 Learning Objectives

After completing this activity, participants should be able to:

- Identify the major components of the Tableau interface
- Distinguish between Dimensions and Measures
- Distinguish between Discrete and Continuous fields
- Convert fields between different data roles
- Use Rows, Columns, Filters, and Marks cards
- Create basic visualizations using drag-and-drop
- Understand how data roles affect visualizations

---

Create an Excel file named **Airbnb_Data.xlsx** using the following data:

# Exercise 1: Explore the Tableau Interface

## Task

Open Tableau Public and connect to the dataset.

### Steps

1. Launch Tableau Public.
2. Connect to **Airbnb_Data.xlsx**.
3. Open **Sheet 1**.

### Activity

Locate and identify the following interface components:

| Component | Purpose |
|------------|----------|
| Data Pane | Displays fields from the dataset |
| Analytics Pane | Contains analytical tools |
| Columns Shelf | Defines the horizontal axis |
| Rows Shelf | Defines the vertical axis |
| Filters Shelf | Filters data |
| Marks Card | Controls appearance |
| Canvas | Displays visualization |
| Toolbar | Provides quick actions |

# Exercise 4: Create Your First Visualization

## Business Question

> Which neighborhood has the highest average room price?

### Steps

1. Drag **Neighborhood** to Columns.
2. Drag **Price** to Rows.

### Expected Output

A bar chart showing average room price by neighborhood.

### Interpretation

- Which neighborhood has the highest average price?
- Which neighborhood has the lowest average price?

# Exercise 5: Understanding Dimensions

## Business Question

> How many listings exist in each neighborhood?

### Steps

1. Create a new worksheet.
2. Rename it **Listings by Neighborhood**.
3. Drag **Neighborhood** to Columns.
4. Drag **Listing ID** to Rows.
5. Change aggregation to **Count**.

### Expected Output

A bar chart showing the number of listings per neighborhood.

### Learning Point

Dimensions are used to group or segment data.

---

# Exercise 6: Understanding Measures

## Business Question

> What is the average price by room type?

### Steps

1. Create a new worksheet.
2. Rename it **Average Price by Room Type**.
3. Drag **Room Type** to Columns.
4. Drag **Price** to Rows.

### Expected Output

Average price for each room type.

### Learning Point

Measures are aggregated automatically.

---

# Exercise 7: Convert a Measure to a Dimension

## Task

Convert "Reviews per Month" into a Dimension.

### Steps

1. Right-click **Reviews per Month**.
2. Select **Convert to Dimension**.

### Questions

Where does the field move?

### Expected Answer

The field moves from Measures to Dimensions.

### Discussion

Why might this be useful?

Example:

- Group listings based on review categories
- Segment listings by review frequency

---

# Exercise 8: Convert a Dimension to a Measure

## Task

Convert "Listing ID" into a Measure.

### Steps

1. Right-click **Listing ID**.
2. Select **Convert to Measure**.

### Questions

What changes?

### Expected Answer

The field moves to the Measures section and can now be aggregated.
