# 📊 Hands-On Activity: Aggregation in Tableau

---

# 🎯 Learning Objectives

After completing this activity, participants should be able to:

- Understand the concept of aggregation
- Apply different aggregation methods to measures
- Aggregate dimensions using Count and Count Distinct
- Compare SUM, AVG, MIN, MAX, and COUNT aggregations
- Convert measures into dimensions for categorical analysis
- Answer business questions using aggregated data
- Build visualizations that summarize information effectively

# Exercise 1: Understanding Tableau's Default Aggregation

## Business Question

> What is the total GDP per Capita represented in the dataset?

### Steps

1. Open Tableau Public.
2. Connect to **Gapminder_Development.xlsx**.
3. Open a new worksheet.
4. Drag **GDP per Capita** to Rows.

### Observe

Tableau automatically displays:

```text
SUM(GDP per Capita)
```

### Discussion

Why does Tableau automatically apply SUM?

### Key Learning

Measures are automatically aggregated when used in a visualization.

# Exercise 2: SUM Aggregation

## Business Question

> Which region has the highest total GDP per Capita?

### Steps

1. Drag **Region** to Columns.
2. Drag **GDP per Capita** to Rows.

### Expected Output

A bar chart displaying:

```text
SUM(GDP per Capita)
```

for each region.

### Interpretation

- Which region has the highest total GDP?
- Which region has the lowest total GDP?

# Exercise 3: Average Aggregation

## Business Question

> Which region has the highest average life expectancy?

### Steps

1. Create a new worksheet.
2. Rename it:

```text
Average Life Expectancy
```

3. Drag **Region** to Columns.
4. Drag **Life Expectancy** to Rows.
5. Click the drop-down menu on Life Expectancy.
6. Select:

```text
Measure → Average
```

### Expected Output

A bar chart showing:

```text
AVG(Life Expectancy)
```

### Discussion

Why might an average be more meaningful than a total?

# Exercise 4: Maximum Aggregation

## Business Question

> Which country has the highest GDP per Capita?

### Steps

1. Create a new worksheet.
2. Drag **Country** to Rows.
3. Drag **GDP per Capita** to Columns.
4. Change aggregation to:

```text

```

When is MAX useful?

# Exercise 8: Aggregating Dimensions

## Business Question

> How many regions exist in the dataset?

### Steps

1. Drag Region to Rows.
2. Click the Region pill.
3. Select:

```text
Measure → Count Distinct
```

### Expected Output

Tableau temporarily treats Region as a measure.

### Learning Point

Dimensions can be aggregated when necessary.

---

# Exercise 9: Converting a Measure into a Dimension

## Business Question

> Can we analyze countries by income group instead of exact GDP values?

### Steps

1. Locate **GDP per Capita**.
2. Right-click.
3. Select:

```text
Convert to Dimension
```

### Expected Result

The field moves to the Dimensions section.

### Discussion

Why convert a measure to a dimension?

Example:

Instead of analyzing:

```text
GDP = 52,000
GDP = 58,000
GDP = 62,000
```

We analyze:

```text
High Income
Upper Middle Income
Lower Middle Income
```
