# Exercise 1: Create a Time Series Chart 📈

## Business Question

> How have births changed over time in Portugal?

### Steps

1. Open Tableau Public.
2. Connect to **Gapminder_Birth_Forecast.xlsx**.
3. Open a new worksheet.
4. Drag **Year** to Columns.
5. Drag **Births** to Rows.
6. Change chart type to **Line Chart**.

### Expected Output

A line chart showing birth trends over time.

---

# Exercise 2: Add a Reference Line 📏

## Business Question

> How do yearly births compare to the average?

### Steps

1. Click **Analytics Pane**.
2. Drag **Reference Line** into the view.
3. Drop it on **Table (Entire View)**.
4. Set value to:

```text
AVG(Births)
```

### Expected Output

A horizontal line representing the average births.

### Interpretation

- Values above the line = above average
- Values below the line = below average

---

# Exercise 3: Customize Reference Line 🎨

## Task

Enhance interpretability.

### Steps

1. Edit Reference Line.
2. Change:
   - Label → “Average Births”
   - Line style → dashed
   - Color → gray

### Discussion

Why are reference lines useful in storytelling?

---

# Exercise 4: Add a Trend Line 📉

## Business Question

> Is there a general upward or downward trend in births?

### Steps

1. Open Analytics Pane.
2. Drag **Trend Line** into the view.
3. Select **Linear Trend Line**.

### Expected Output

A trend line over the birth data.

### Interpretation

- Upward slope → increasing trend
- Downward slope → decreasing trend

---

# Exercise 5: Compare Multiple Trends 🌍

## Business Question

> How do Portugal and Japan compare over time?

### Steps

1. Drag **Country** to Color.
2. Ensure **Year** is on Columns.
3. Keep **Births** on Rows.
4. Add **Trend Line**.

### Expected Output

Separate trend lines per country.

---

# Exercise 6: Forecasting Future Values 🔮

## Business Question

> What will birth values look like in the future?

### Steps

1. Ensure Year is continuous (green field).
2. Open **Analytics Pane**.
3. Drag **Forecast** into the view.

### Expected Output

A forecast extending beyond available years.

---

# Exercise 7: Understanding Forecast Output 📊

## Observation

Forecast includes:

- 📊 Predicted values
- 📉 Confidence intervals (upper & lower bounds)
- 🔮 Future projection

### Discussion

Why do forecasts include uncertainty ranges?
