# 📊 Hands-On Activity: Dashboards and Stories in Tableau

---

# 🎯 Learning Objectives

After completing this activity, participants should be able to:

- Build and organize Tableau worksheets
- Create interactive dashboards
- Use filters and actions across multiple views
- Understand dashboard layout techniques
- Build a Story using multiple story points
- Combine worksheets and dashboards into a narrative
- Communicate insights effectively using Tableau

# 🧩 PART 1: Build Worksheets

---

## Exercise 1: Sales Trend by Year 📈

### Business Question
> How have video game sales changed over time?

### Steps
1. Open Tableau Public
2. Connect to **VideoGame_Dashboard_Story.xlsx**
3. Drag **Year → Columns**
4. Drag **Sales → Rows**
5. Change to **Line Chart**

---

## Exercise 2: Sales by Platform 🎮

### Business Question
> Which platform generates the highest sales?

### Steps
1. Drag **Platform → Columns**
2. Drag **Sales → Rows**
3. Change to **Bar Chart**
4. Sort descending

---

## Exercise 3: Genre Performance 📊

### Business Question
> Which genre performs best?

### Steps
1. Drag **Genre → Columns**
2. Drag **Sales → Rows**
3. Add **Color → Genre**
4. Sort descending

---

## Exercise 4: Platform vs Genre Heatmap 🔥

### Business Question
> Which platform performs best per genre?

### Steps
1. Drag **Platform → Rows**
2. Drag **Genre → Columns**
3. Drag **Sales → Color**
4. Change Marks → Square

---

# 📋 PART 2: Build Dashboard

---

## Exercise 5: Create Dashboard 📋

### Steps
1. Click **New Dashboard**
2. Add worksheets:
   - Sales Trend
   - Platform Performance
   - Genre Performance
   - Heatmap

---

## Dashboard Setup

### 🎯 Title

```text
Global Video Game Performance Dashboard
```

---

## Exercise 6: Add Interactivity 🎛️

### Add Filters
1. Drag **Year → Filters**
2. Show Filter

### Add Platform Filter
1. Drag **Platform → Filters**
2. Show Filter

---

## Exercise 7: Add Dashboard Actions 🔗

### Steps
1. Go to **Dashboard → Actions**
2. Add:
   - Filter Action
   - Highlight Action

### Behavior
Clicking a platform updates all charts.

---

## Exercise 8: Improve Layout 🎨

### Steps
- Arrange charts in grid format
- Use containers (vertical/horizontal)
- Add spacing between visuals
- Add titles for each section
