# 📊 Hands-On Activity: Filtering and Sorting in Tableau

---

# 🎯 Learning Objectives

After completing this activity, participants should be able to:

- Apply Dimension Filters
- Apply Measure Filters
- Create interactive filters
- Sort visualizations in ascending and descending order
- Use Top and Bottom filters
- Analyze filtered and sorted data
- Answer business questions using Tableau

---

# 📁 Dataset

Create an Excel file named **Gapminder_Communication.xlsx** using the following data:

| Country | Region | Mobile Phones per 100 People | Broadband Subscribers per 100 People | Population (Millions) |
|----------|----------|----------:|----------:|----------:|
| United States | North America | 110 | 35 | 335 |
| Canada | North America | 105 | 38 | 40 |
| Germany | Europe | 115 | 42 | 84 |
| France | Europe | 112 | 40 | 66 |
| Japan | Asia | 125 | 44 | 124 |
| South Korea | Asia | 130 | 47 | 52 |
| Philippines | Asia | 95 | 12 | 115 |
| Indonesia | Asia | 90 | 8 | 280 |
| Brazil | South America | 98 | 15 | 216 |
| Argentina | South America | 102 | 20 | 46 |

# Exercise 1: Create a Basic Visualization

## Business Question

> Which countries have the highest number of mobile phones per 100 people?

### Steps

1. Open Tableau Public.
2. Connect to **Gapminder_Communication.xlsx**.
3. Open a new worksheet.
4. Rename it **Mobile Phone Usage**.
5. Drag **Country** to Columns.
6. Drag **Mobile Phones per 100 People** to Rows.

### Expected Output

A bar chart showing mobile phone ownership by country.

### Discussion

- Which country has the highest value?
- Which country has the lowest value?

---

# Exercise 2: Dimension Filter

## Business Question

> How does communication infrastructure vary by region?

### Steps

1. Drag **Region** to the Filters shelf.
2. Select only **Asia**.
3. Click **OK**.
4. Right-click the filter.
5. Select **Show Filter**.

### Activity

Switch between:

- Asia
- Europe
- North America
- South America

### Discussion

How does filtering help focus analysis?

---

# Exercise 3: Multiple Dimension Filters

## Business Question

> Can we compare selected countries only?

### Steps

1. Drag **Country** to Filters.
2. Select:
   - Philippines
   - Japan
   - South Korea

3. Click **OK**.

### Expected Output

The chart displays only the selected countries.

### Discussion

Why might decision-makers use this type of filter?

---

# Exercise 4: Measure Filter

## Business Question

> Which countries have at least 100 mobile phones per 100 people?

### Steps

1. Drag **Mobile Phones per 100 People** to Filters.
2. Select **At Least**.
3. Enter:

```text
100
```

4. Click **OK**.

### Expected Output

Countries with values below 100 are excluded.

### Discussion

Which countries remain after filtering?

---

# Exercise 5: Range Filter

## Business Question

> Which countries have broadband subscribers between 15 and 40 per 100 people?

### Steps

1. Drag **Broadband Subscribers per 100 People** to Filters.
2. Select **Range of Values**.
3. Enter:

| Minimum | Maximum |
|----------|----------|
| 15 | 40 |

### Expected Output

Only countries within the specified range are displayed.

### Discussion

Why are range filters useful?

# Exercise 7: Sort by Measure Values

## Business Question

> Which countries have the highest mobile phone penetration?

### Steps

1. Click the **Sort Descending** icon on the toolbar.

OR

1. Right-click Country.
2. Select **Sort**.
3. Sort by:
   - Field
   - Mobile Phones per 100 People
   - Descending

### Expected Output

Countries are ranked from highest to lowest.

### Discussion

Why is this sorting method more useful?

---

# Exercise 8: Top N Filter

## Business Question

> Which are the Top 5 countries in mobile phone ownership?

### Steps

1. Drag **Country** to Filters.
2. Open the **Top** tab.
3. Select:

```text
Top 5 by SUM(Mobile Phones per 100 People)
```

4. Click OK.

### Expected Output

Only the top five countries remain.

### Discussion

Who might use Top N analysis?

Examples:

- Executives
- Policymakers
- Researchers

---

# Exercise 9: Bottom N Filter

## Business Question

> Which countries have the lowest communication infrastructure indicators?

### Steps

1. Edit the Country filter.
2. Open the Top tab.
3. Select:

```text
Bottom 3 by SUM(Broadband Subscribers per 100 People)
```

### Expected Output

The three lowest-performing countries appear.

### Discussion

Why is identifying low-performing areas important?

# Exercise 12: Communication Infrastructure Analysis

## Business Question

> Which countries appear most digitally developed?

### Steps

1. Create a scatter plot.

2. Drag:

   - Mobile Phones per 100 People → Columns
   - Broadband Subscribers per 100 People → Rows

3. Drag:
   - Country → Label
   - Region → Color

### Expected Output

A scatter plot comparing mobile and broadband adoption.

### Discussion

Which countries appear in the upper-right corner?

What does this indicate?
