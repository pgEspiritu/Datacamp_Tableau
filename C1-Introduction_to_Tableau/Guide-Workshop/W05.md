# Exercise 1: Create Your First Calculated Field

## Business Question

> What is the average years of schooling for young adults (15–24)?

### Steps

1. Open Tableau Public.
2. Connect to **Gapminder_Education.xlsx**.
3. In the Data Pane, right-click anywhere.
4. Select:

```text
Create Calculated Field
```

5. Name the field:

```text
Average Schooling 15-24
```

6. Enter the formula:

```text
([Male 15-24] + [Female 15-24]) / 2
```

7. Click **OK**.

### Expected Result

A new calculated field appears in the Data Pane.

### Discussion

Why is this considered a virtual column?

# Exercise 3: Create a Gender Gap Calculation

## Business Question

> How large is the education gap between young males and females?

### Steps

Create a calculated field named:

```text
Gender Gap 15-24
```

Formula:

```text
[Female 15-24] - [Male 15-24]
```

### Expected Output

Positive values indicate females have more years of schooling.

Negative values indicate males have more years of schooling.

### Discussion

What does a value of 0 mean?

# Exercise 4: Create an Overall Average

## Business Question

> What is the overall average years of schooling across all age groups?

### Steps

Create a calculated field named:

```text
Overall Average Schooling
```

Formula:

```text
(
[Male 15-24] +
[Female 15-24] +
[Male 25-34] +
[Female 25-34] +
[Male 35-44] +
[Female 35-44] +
[Male 45-54] +
[Female 45-54] +
[Male 55+] +
[Female 55+]
) / 10
```

### Expected Output

A single measure representing the overall average education level.

---

# Exercise 5: Using the ROUND Function

## Business Question

> Can we display averages without decimals?

### Steps

Create a calculated field:

```text
Rounded Average Schooling
```

Formula:

```text
ROUND([Overall Average Schooling],0)
```

### Expected Output

Values are rounded to the nearest whole number.

### Example

| Original | Rounded |
|----------|----------|
| 9.7 | 10 |
| 8.2 | 8 |
| 11.6 | 12 |

# Exercise 6: Categorize Countries

## Business Question

> Which countries have high or low educational attainment?

### Steps

Create a calculated field named:

```text
Education Category
```

Formula:

```text
IF [Overall Average Schooling] >= 10 THEN
"High"
ELSE
"Low"
END
```

### Expected Output

Each country is assigned a category.

| Country | Category |
|----------|----------|
| Malaysia | High |
| Thailand | High |
| Philippines | Low |

### Discussion

Why are categorical fields useful?

# Exercise 8: Percentage Difference Calculation

## Business Question

> What percentage advantage do females have over males in the 15–24 age group?

### Steps

Create a calculated field:

```text
Female Advantage (%)
```

Formula:

```text
(([Female 15-24] - [Male 15-24]) / [Male 15-24]) * 100
```

### Expected Output

Percentage difference between females and males.

### Interpretation

Positive percentage:

```text
Females have higher schooling levels.
```

Negative percentage:

```text
Males have higher schooling levels.
```
