# 🎨 Best Practices in Data Visualization

Welcome back!

Now that you're familiar with both **basic** and **advanced** charts in **Tableau**, it's time to focus on the **best practices** that make visualizations clear, trustworthy, and impactful.

---

# ⚠️ Consequences of Bad Data Visualization

No data analyst intends to mislead people with data.

While calculation errors are often the primary source of mistakes, poor visualization choices can also lead to serious consequences.

Poor visualizations can result in:

- 😕 Unattractive charts that reduce stakeholder interest and trust
- ❓ Confusing visuals that raise more questions than answers
- 🚨 Misleading interpretations of data
- 💼 Poor business decisions based on incorrect conclusions

Good visualization is not just about making charts look nice—it's about communicating information accurately.

---

# 🚫 Common Mistakes in Data Visualization

Several common issues can reduce the effectiveness of a visualization.

### 📊 Poor Chart Selection

Choosing a chart that doesn't fit the data or the analytical question.

### 🎨 Misleading Chart Design

Improper chart layouts or misleading axis scales.

### 🌈 Incorrect Use of Color, Shape, or Size

Visual elements that distract rather than clarify.

### 🖌️ Neglecting Formatting

Missing titles, labels, or inconsistent formatting.

### ❌ Poor Handling of Missing Data

Ignoring null values can create misleading conclusions.

# 📈 Choose the Right Chart for the Right Use Case

Before creating any visualization, ask yourself several important questions.

## 🤔 What are you trying to show?

Examples include:

- 🔄 Relationships
- 📊 Comparisons
- 📦 Distributions
- 🧩 Composition
- 📈 Trends over time

## 📂 What type of data do you have?

Consider:

- Number of variables
- Number of categories
- Time series data
- Absolute vs. relative values

## 👥 Who is your audience?

Always think about:

- Their level of data literacy
- The message you want to communicate
- The action you expect them to take

For example:

- 📋 Senior executives may prefer a concise dashboard highlighting key metrics.
- 🔍 Analysts may benefit from more detailed and interactive visualizations.

# 🏷️ Design Chart Elements Correctly

A well-designed chart should clearly explain itself.

Always include:

- 📝 Descriptive chart title
- 🏷️ Axis titles
- 📌 Data labels (when appropriate)
- 📏 Clearly labeled axes

### ⚠️ Be Careful with Axis Scaling

Changing the Y-axis range can exaggerate or minimize trends.

Whenever appropriate:

- ✅ Start the axis at **0**
- ✅ Avoid misleading scale adjustments

A chart should present data honestly, not manipulate perception.

---

# 🌈 Use Color Intentionally

Color is one of the most powerful visual tools—but only when used thoughtfully.

## ✅ Best Practices

- 🎯 Highlight important data points
- 🎨 Maintain consistent colors throughout dashboards
- 🏢 Follow organizational or brand color guidelines

## ⚠️ Avoid

- 🔴 Overusing red
- 🟡 Excessive amber/yellow
- 🟢 Overusing green

These colors typically imply:

- Good vs. bad
- Increase vs. decrease
- Status indicators

If color doesn't add information, it's often better not to use it.

---

# 📐 Use Shape and Size Carefully

Shape and size allow additional dimensions to be represented in a chart.

For example:

- 🌍 Bubble size can represent population.
- 🎨 Color can group countries by continent.

This enables multiple variables to be displayed simultaneously.

### ⚠️ However...

Avoid using shapes or varying sizes when:

- 🎯 Precise values are important
- 😵 They distract from the main message
- 📊 They make comparisons difficult

Extra visual elements should improve understanding—not reduce it.

---

# ❓ The Story Behind Missing Values (Nulls)

Good visualization isn't only about appearance—it's also about accurately representing the data.

Ignoring **null values** can unintentionally hide important information.

### Example

A sales chart may appear to show continuous daily revenue.

However, once null values are included, you may discover that:

- 📅 Some days had **no sales at all**

Excluding missing values can create an incomplete or overly optimistic story.

Always understand why values are missing before deciding how to handle them.
