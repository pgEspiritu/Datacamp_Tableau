# 📊 More Advanced Visualizations

Now that we've learned how to build basic charts in **Tableau**, it's time to explore more advanced visualizations that provide deeper insights and answer more complex business questions.

---

# 🎯 Visualizing for Various Use Cases

Different analytical questions require different chart types.

For many scenarios, there is more than one suitable visualization. In this chapter, we'll explore several advanced charts that require a bit more effort to create but offer powerful ways to communicate data.

Let's examine them one by one.

---

# 📦 Box Plot — Understanding Data Distribution

A **box plot** is a standardized chart used to display the distribution of numerical data based on a **five-number summary**.

It displays:

- 📉 Minimum
- 📊 First Quartile (Q1)
- ➖ Median
- 📈 Third Quartile (Q3)
- 📌 Maximum

The lines extending from the box are called **whiskers**.

## ✅ Best Used For

- 📊 Understanding data distribution
- 📈 Measuring variance
- ⚖️ Detecting symmetry or skewness
- 🚨 Identifying potential outliers

Although box plots are excellent for distribution analysis, **histograms** are another common way to visualize distributions.

---

# 🌊 Waterfall (Bridge) Chart — Showing Contributions and Change

A **waterfall chart**, also known as a **bridge chart**, explains how individual categories contribute to the overall change between two values.

It typically begins at a baseline of **zero**, followed by bars representing positive and negative contributions.

Positive and negative values are usually distinguished using different colors.

## ✅ Best Used For

- 💰 Financial reporting
- 📈 Profit and loss analysis
- 📊 Explaining how individual categories contribute to a total
- 🔍 Breaking down aggregated values

### 👍 Advantages

- Makes overall changes easy to understand.
- Reveals hidden gains and losses within totals.
- Clearly separates positive and negative contributions.

### ⚠️ Limitation

- Works best with a relatively small number of categories.

---

# 🔥 Heat Map — Density and Matrix Comparisons

A **heat map** uses color intensity to represent values, making it easy to identify patterns, density, and comparisons.

Heat maps come in several forms.

Common examples include:

- 🌍 Geographic heat maps (e.g., population density)
- 🖱️ Website click maps
- 📊 Matrix comparisons

## ✅ Best Used For

- 📈 Showing density
- 🔍 Comparing many values simultaneously
- 🎯 Quickly identifying patterns
- 📋 Large matrix comparisons

### 👍 Advantages

- Instantly highlights trends through color gradients.
- Provides a strong high-level overview.

### ⚠️ Limitation

- Without labels, precise interpretation can be difficult.

---

# ⚪ Scatter Plot — Relationships and Correlations

A **scatter plot** displays the relationship between **two numerical variables** using points plotted along the **X-axis** and **Y-axis**.

Additional dimensions can be represented using:

- 🎨 Color
- 📏 Size
- 🔷 Shape

Scatter plots can contain thousands of data points while remaining effective.

## ✅ Best Used For

- 🔍 Exploratory Data Analysis (EDA)
- 📈 Identifying correlations
- 🎯 Quadrant analysis
- 📊 Comparing two numerical measures

### 👍 Advantages

- Excellent for detecting relationships between variables.
- Supports multiple dimensions.
- Handles large datasets well.

### ⚠️ Limitation

- Can become cluttered with many data points.
- Weak or nonexistent correlations may make insights difficult to interpret.
